Continuous Random Variables

Let S be the sample space associated with some experiment E. A random variable X is a function that assigns a real number X(s) to each elementary event s∈S. In this case, as the outcome of the experiment, i.e. the specific s, is not predetermined, then the value of X(s) is not fixed. This means that the value of the random variable is determined by the specific outcome of the experiment.

A continuous random variable can take an uncountably infinite number of values.
