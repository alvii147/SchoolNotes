Stability

Consider a system with transfer function H(s) such that,

H(s)=N(s)D(s)

where

D(s)=sn+an−1sn−1+...+a1s+a0=(s+p1)(s+p2)...(s+pn)

The system is stable if and only if all roots of D(s), (p1,p2...pn) are in the open left half plane. In order words,

Re(pi)<0,for all0<i≤n

Necessary Condition for Stability

A system defined by transfer function H(s) is stable only if all coefficients of D(s), (a0,a1...an−1) are positive. In other words, if

H(s)=N(s)D(s)

and

D(s)=sn+an−1sn−1+...+a1s+a0=(s+p1)(s+p2)...(s+pn)

Then,

ai>0for all0<i≤n⟹the system is stable
